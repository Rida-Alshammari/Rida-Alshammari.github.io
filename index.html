Projects

Throughout my journey, I have worked on several technical projects, and I will be showcasing some of them, including my final graduation project, which focuses on Violence Detection.

Project Summary

This project aims to classify visual content as either violent or non-violent, helping to determine its suitability for children and sensitive users. The project falls under the fields of monitoring and security, as it detects content and identifies potential risks using artificial intelligence and machine learning techniques.

The project relies on Microsoft Azure for image analysis. However, since Azure does not support videos, a video classification model was developed using Python and machine learning to process videos and integrate them with the website.

  • Artificial intelligence (AI)
  • Microsoft Azure
  • Python
  • HTML & CSS
Download

System Operation

  • The user uploads an image or video via the website.
  • The content is analyzed using an AI and machine learning model.
  • The website offers an option to retest another piece of content.
  • The analysis result is displayed as a percentage indicating the level of violence in the content.

Project Objectives

  • To protect viewers and enhance a safer viewing environment.
  • To determine the suitability of content for children and sensitive users.
  • To enhance digital security and provide intelligent content monitoring.
  • To offer an effective tool for content providers, parents, and educational platforms in safeguarding users.
